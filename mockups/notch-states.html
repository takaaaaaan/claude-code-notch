Claude Notch for Windows — UIモックアップ

画面上部中央のノッチ風オーバーレイ。Claude Code の Hooks がローカルポートへ送ったイベントに反応します。
下のボタンで各状態を切り替えて、見た目とアニメーションを確認してください。

キャラが変化する状態（軽いイベント）
通知カードが出る状態（重要イベント）
待機中 … キャラがゆっくり呼吸。接続中ドットが点灯。
ツール実行中 … キャラが青くなり回転気味にパルス（Bash/編集など作業中）。
サブエージェント稼働 … キャラが紫に、速めにバウンド。
タスク完了 … 緑バッジのカードがニュッと展開。
許可待ち … 黄バッジ＋キャラが小刻みに揺れて注意喚起。
⚠️ キャラの絵は仮（絵文字）です。 本番では Claude Code のキャラクター画像（表情違い数種）に差し替える前提です。色・動きの方向性を見てください。

仕組み：Claude Code 側の Hooks（Stop / Notification / PreToolUse / SubagentStop）が localhost:ポート へ JSON を POST → このオーバーレイが受信して表示。
